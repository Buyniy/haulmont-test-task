Test Task
=========

Prerequisites
-------------

* [Java Development Kit (JDK) 8](https://www.oracle.com/java/technologies/downloads/#java8)
* [Maven 3](https://maven.apache.org/download.cgi)

Build and Run
-------------

1. Run in the command line:
	```
	mvn package
	mvn jetty:run
	```

2. Open `http://localhost:8080` in a web browser.
